üßπ Data Cleaning (SQL)

Removing Duplicate Records
WITH duplicate_cte AS ( SELECT *, ROW_NUMBER() OVER ( PARTITION BY company, location, industry, total_laid_off, percentage_laid_off, date, stage, country, funds_raised_millions ) AS row_num FROM layoff_staging ) DELETE FROM layoff_staging2 WHERE row_num > 1;

Duplicates were identified and removed using window functions.

Standardizing Industry Names
UPDATE layoff_staging2 SET industry = 'Crypto' WHERE industry IN ('Crypto Currency', 'CryptoCurrency');

Inconsistent industry labels were standardized.

Fixing Date Format
UPDATE layoff_staging2 SET date = STR_TO_DATE(date, '%m/%d/%Y'); ALTER TABLE layoff_staging2 MODIFY date DATE;

Date values were converted from text to SQL DATE format.

üìä Exploratory Data Analysis

Top 10 Companies by Layoffs
SELECT company, SUM(total_laid_off) AS total_layoffs FROM layoff_staging2 GROUP BY company ORDER BY total_layoffs DESC LIMIT 10;

Identifies companies responsible for the highest number of layoffs.

Industry-wise Layoffs
SELECT industry, SUM(total_laid_off) FROM layoff_staging2 GROUP BY industry ORDER BY 2 DESC;

Highlights industries most impacted by layoffs.

Country-wise Layoffs
SELECT country, SUM(total_laid_off) FROM layoff_staging2 GROUP BY country ORDER BY 2 DESC;

Shows geographical distribution of layoffs.

Year-wise Layoff Trends
SELECT YEAR(date), SUM(total_laid_off) FROM layoff_staging2 GROUP BY YEAR(date) ORDER BY 1 DESC;

Analyzes how layoffs changed over time.
